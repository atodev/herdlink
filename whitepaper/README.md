# White paper

*Herd-Relative and Social-Network Anomaly Detection for Livestock Wearables: An In-Silico Study*

A peer-review-style write-up of the HerdLink method and its simulation evaluation.
**All results are in-silico** — see the Limitations section of the paper.

## Building

The numbers, tables, and figures are generated by the evaluation harness, not
hand-typed. Regenerate them, then compile:

```bash
# from the repo root — runs 30 seeded replicates (~a few minutes),
# writes whitepaper/data/*.{tex,csv,dat} and results.json
npm run evaluate

# then, with a TeX distribution installed:
cd whitepaper
make            # -> main.pdf
```

Reproducibility: the harness is fully seeded (base seed in `data/config.tex`),
so `npm run evaluate` twice produces byte-identical `data/results.json`.

To run a quick, lower-variance pass while iterating:

```bash
REPLICATES=5 npm run evaluate
```

## Layout

- `main.tex` — the paper. `\input`s everything in `data/` so no result is transcribed.
- `references.bib` — bibliography.
- `data/` — harness output (git-ignored; regenerate with `npm run evaluate`):
  - `config.tex` — run configuration + headline-number macros
  - `expA_classification.tex` — held-out precision/recall
  - `expB_fp.{tex,csv,dat}` — false-positive rate by weather regime × feature family
  - `expC_latency.{tex,csv,dat}` — detection latency by condition × feature family
  - `results.json` — everything, machine-readable

## Harness

`scripts/evaluate.ts` (repo root). Experiments: A classification, B false
positives under weather, C detection latency, D feature-family ablation. It
reuses the production simulator (`src/sim/herd.ts`), feature extraction
(`src/sim/features.ts`), social analytics (`src/sim/social.ts`), the live
detector (`src/sim/analytics.ts`), and the trainer (`src/sim/trainer.ts`) so the
paper measures the same code the demo runs.

## No TeX installed?

The sources are complete and standard (article, natbib, pgfplots). Any TeX
distribution (TeX Live, MacTeX, Overleaf) will compile `main.tex` after
`npm run evaluate` has populated `data/`.
